# Calculadora em Python

Este projeto representa meu primeiro contato prático com Python.  
Foi desenvolvido com o objetivo de praticar lógica de programação, estruturas condicionais e o uso do laço `while`.

O código foi mantido de forma simples e com limitações intencionais, para que seja possível acompanhar minha evolução ao longo do aprendizado.

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão
- Validação de divisão por zero

## Tecnologias
- Python 3
